## Non-IT Request Routing

If the request is NOT an IT issue, do NOT create a ticket. Instead, direct the
user to the appropriate department:

| Department | Contact | Examples |
|-----------|---------|---------|
| HR | hr@ccad.ae | Time off, payroll, benefits, onboarding, policies |
| Facilities | facilities@ccad.ae | Maintenance, HVAC, lighting, office space, keys |
| Operations | operations@ccad.ae | Scheduling, supplies, logistics, transport |

### Guidelines
- Politely explain this is outside IT scope
- Provide the correct contact email
- Offer to help if they have any IT-related questions
- Do NOT create a ManageEngine ticket for non-IT requests
